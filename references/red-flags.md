# Red Flags

Use red flags to decide what to investigate, not as proof by themselves. A red flag raises the burden of evidence; it does not replace evidence.

## Common Pseudoscience Terms

- 量子, 纳米, 磁场, 远红外, 负离子, 能量, 频率
- 排毒, 酸碱体质, 激活细胞, 提高自愈力, 增强免疫力
- 祖传, 宫廷, 千年古方, 秘方
- 纯天然所以无副作用
- NASA, 诺贝尔奖, 海外认证, 军工技术 without a verifiable official link

## Marketing Patterns

- Cures many unrelated conditions.
- Promises a large effect with no dose, population, or timeframe.
- Relies on testimonials, before-after photos, celebrity use, or user stories.
- Uses urgency tactics: limited stock, last chance, only today.
- Uses expensive pricing for common cheap ingredients.
- Claims to replace evidence-based medical care.
- Hides exact ingredients, concentration, model specifications, certification numbers, or test methods.

## Evaluation Response

When a red flag appears:

- Translate the wording into a standard scientific concept if possible.
- If no standard concept exists, mark that mechanism 🔴.
- If a plausible concept exists, still require human/consumer evidence before a positive verdict.
- Explain the issue neutrally; do not turn the answer into a rant.
